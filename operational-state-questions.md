# Operational State Questions: Cloudsploitable
This document lists the main questions that should be answerable quickly from the OE dashboard, which is meant to give a quick view of current service health, recent activity, and possible warning signs before any deeper analysis begins.

## Main Questions
- Is the service currently healthy?
- Is average response time still within a normal range?
- Is storage usage becoming a problem?
- How many sessions are currently active?
- How many users are currently blocked after repeated failed login attempts?
- Are successful and failed authentication attempts trending normally?
- Are error responses increasing, and which error types are appearing most often?
- Do recent operational events show anything suspicious, unusual, or helpful for understanding the current state?
